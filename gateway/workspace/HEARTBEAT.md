# Heartbeat — стоячие инструкции

При каждом heartbeat-тике выполни проверку. Если ничего не требует внимания — ответь HEARTBEAT_OK.

## Проверки
- Проверь срочные письма (is:important is:unread newer_than:1h) на всех аккаунтах. Если есть — уведоми пользователя через Telegram с кратким саммари.
- Проверь календарь: если через 15 минут или меньше есть встреча — напомни.

## Правила
- Не повторяй уведомления, которые уже были отправлены в этой сессии.
- Этот файл можно обновлять по запросу пользователя. Если он попросит добавить или убрать проверку — отредактируй HEARTBEAT.md.
